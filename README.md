"# Assignment_4" 

Description:
This app is an ebay scrapping tool. Its use is simple, in the search bar write the name of the
object you are searching for and click enter. From there the application will return the 6 cheapest
options available, with the option to filter the for cheapest with free shipping, cheapest 
with shipping cost included, etc.
